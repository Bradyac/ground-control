Technologies Used

Front-End

Built with Angular, the front-end is a SPA that utilizes modules HTTPClient, NgxPagination, Clipboard, Routing and custom components/services. The design was made to be simple, clean and uniform across the site. Content boxes are made to appear like glass, allowing the details of the applications background to come through. Each page has some different variation of the glass styled content boxes making them all unique yet uniform.


Technologies Used: Angular (Typescript, HTML5, CSS3)

Back-End

Data for the site is fetched from a public API and is stored in a NoSQL database. Then on request from the front-end the data is fetched from the database via serverless Lambda functions.

Data is not directly loaded from the public API to the front-end to reduce the amount of requests to the API. New/updated data is loaded into the database daily and requests to the database from the front-end are unlimited.


Technologies Used: Netlify (Hosting+Functions), Node.js/Express (JavaScript), MongoDB/Mongoose, Heroku Scheduler

The Data

The Space Devs

The Space Devs is a group of space enthusiast developers working on a range of services, united in a common goal to improve public knowledge and accessibility of spaceflight information. They are the providers of both public FREE APIs used in this application. To learn more click the link below to visit their website!

Launch Library

The Launch Library is a public API that tracks all orbital and popular suborbital rocket launches. This service provides the details on upcoming rocket launches throughout the application.

SNAPI

The Spaceflight News API, or SNAPI for short, is a public API for providing regularly updated space related news. This service provides the articles found on the home page.
