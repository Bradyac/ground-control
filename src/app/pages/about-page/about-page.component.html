About

The Site

Ground Control is a simple web app for viewing rocket launches and current space related news! Check out upcoming/past rocket launches on the launches page. Keep up to date with what is happening in the space industry on the news page.

Technologies Used

Front-End

The front end of this web app is what you see right now. It was built using the Angular framework.

Tecnologies Used: Angular, Typescript, HTML5, CSS3.

Back-End

The server side application that the front-end communicates with is built using Spring Boot.

Technologies Used: Spring Framework (Spring Boot) and Java.
